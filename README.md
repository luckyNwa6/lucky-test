# Vue2 测试

## 🐑 分支

main------------>vue2 的各种包和组件测试

testVue3-------->vue3 的各种包和组件测试

## 🐶 测试环境

node 18.16.1

包管理使用 yarn

推荐独立一个文件夹专门存测试项目的

node 的测试需要 18 折腾启动

```
npm install node-gyp -g

npm config set registry https://registry.npmjs.org

npm uninstall node-sass

npm install sass

```
